Trang web tính chu vi và diện tích hình tròn

Công thức tính chu vi hình tròn C = 2 x R x π.Trong đó

  • C là chu vi hình tròn
  • R là bán kính hình tròn
  • π là hằng số Pi (giá trị tương đương 3,14)

Công thức tính diện tích hình tròn S = R^2 x π.Trong đó:

  • S là diện tích hình tròn
  • R là bán kính hình tròn
  • π là hằng số Pi (giá trị tương đương 3,14)
